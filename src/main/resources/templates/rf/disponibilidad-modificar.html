RF6 - Modificar Disponibilidad

Modifica una disponibilidad existente

✗ Error:
ID de la disponibilidad que deseas modificar
Formato: HH:MM-HH:MM (Ejemplo: 14:00-18:00)

ℹ️ Información

  • Primero debes tener el ID de la disponibilidad que deseas modificar
  • No pueden haber franjas horarias superpuestas para el mismo día
  • Puedes cambiar el día, la franja horaria o el tipo de transporte
  • La disponibilidad debe existir en el sistema
